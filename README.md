# Music Generator

This project is developed as part of the CodeAlpha AI Internship.

## Description
The Music Generator creates random simple melodies using AI logic in Python. Users can click a button to generate a new sequence of music notes.

## Technologies Used
- Python
- Streamlit
- Random module (for melody generation)

## Features
- Generates random melodies
- Interactive interface
- Easy to use

## How It Works
1. User clicks the "Generate Music" button
2. The system generates a sequence of musical notes
3. The notes are displayed on the screen
